# ECOBITE FOODS 🌿

Aplicación móvil desarrollada con React Native y Expo para la venta de snacks saludables basados en superfoods andinos.

## 🛠️ Configuración del Entorno de Desarrollo

Requisitos previos:
* Tener [Node.js](https://nodejs.org/) instalado.
* Tener la aplicación **Expo Go** instalada en tu celular o un emulador en la PC.

## 🚀 Pasos para Instalar

1. Abre una terminal y clona este repositorio:
   git clone https://github.com/LuisMallma/ecobite-foods.git

2. Ingresa a la carpeta del proyecto:
   cd SnacksAndinos

3. Instala las dependencias:
   npm install

## 📱 Ejecutar y Probar la Aplicación

1. Inicia el servidor de Expo ejecutando este comando:
   npx expo start

2. Para probar la app:
   * **En celular:** Escanea el código QR que aparece en la terminal usando la app Expo Go.
   * **En emulador:** Presiona la tecla `a` en la terminal para abrir en Android.
